How to create free maps for your Garmin GPS unit

I've put together a number of different pages on this topic, which covers how to take data from OpenStreetMap and turn it into a Garmin file which you can load into your GPS unit. If you've never done this before, I'd suggest working your way through them in order.
  1. Basic step-by-step guide to map creation - A newbie guide
  2. Guide to mkgmap style files - How to customise styles
  3. Guide to TYP files - What they are, how to edit them and use them in mkgmap
  4. Comprehensive list of mkgmap/splitter options and other goodies - This is a "scrapbook" where I keep any nuggets of info I discover on the mkgmap mailing list or elsewhere. Updated:1/2/2010
  5. FAQ
  6. Advanced guide to splitter
  7. Walk through my compilation process step-by-step
  8. Improve OSM data using mkgmap output
  9. Downloads Links to various customised files for download Updated:13/2/2010
  10. Screenshots of my style/TYP files in action
  11. Screenshots & IMG downloads of default Garmin types

These pages are very much work in progress. I wrote them to try to help newcomers to mkgmap, because whilst there's a lot of information for it, it's not particularly well organised. Please please please send any suggestions or comments to help me improve things.
