Info
====

Il file è stato creato partendo da dati di OpenStreetMap (www.openstreetmap.org) utilizzando ital.img (https://github.com/lucadelu/ital.img)

The file is created by ital.img (https://github.com/lucadelu/ital.img) using OpenStreetMap data (www.openstreetmap.org)

How to use
==========

Per utilizzare la mappa sul Garmin basta copiare il file sulla scheda di memoria all'interno della cartella garmin, se non avete suddetta cartella createla. Al riavvio del GPS avrete la cartografia sul vostro strumento.

To use the map on your Garmin, copy the file in the microSD card inside garmin folder. Restart GPS and you found the new cartography on your GPS

Attention
----------
La mappa non funziona su programmi tipo QlandKarte o MapSource perchè è stata creata solamente per i GPS

The map doesn't work on software like QlandKarte or MapSource, it is created only for GPS with flash card.
